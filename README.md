# Sistemas WEB I 
Esse é um projeto destinado ao curso de desenvolvimento de sistemas web I. O tema do trabalho é sobre controle de revisão carros. 

## Resumo 
Controle de Revisão de Veículo o sistema deve possibilitar o usuário a Cadastrar veículos e suas informações básicas marca, modelo, ano, placa cor quilometragem etc.; registrar cada revisão feita, com data, tipo de serviço, custo e responsável; controlar próximas revisões com base na data ou na quilometragem; emitir relatórios de histórico de manutenção; possibilitar o acompanhamento dos gastos com serviços e peças. 

## 1. Tema 
O trabalho final tem como tema o desenvolvimento de um sistema web de Controle de Revisão de veículos, proporcionando um fácil meio de acompanhamento dos veículos. 

## 2. Escopo 
Este projeto terá as seguintes funcionalidades: 

 #### Cadastrar Veículos  

 Cadastras os veículos com os seguintes parâmetros: 

- Marca 

- Modelo 

- Ano 

- Placa 

- Cor 

- Quilometragem atual 

#### Editar/Deletar 
Deve permitir: 
- Editar revisão cadastrada 
- Excluir revisão cadastrada 
- Deletar veículos cadastrados 
- Editar veículos cadastrados 

#### Relatórios e Histórico de Manutenção 
O sistema deve fornecer informações consolidadas sobre as revisões já realizadas. 
- Histórico completo do veículo com todas as revisões 
Relatórios filtrados por: 
- Período (mensal, anual) 
- Tipo de serviço 
- Responsável

## 3. Restrições 
Neste trabalho não serão considerados cadastro de novos usuários do sistema, Emissão de relatórios em pdf/Excel. (As restrições estão suscetíveis a mudanças) 

## 4. Protótipo 
Serão elaboradas as seguintes páginas

#### Dashboard
Mostra um resumo geral do sistema. Exibe quantos veículos estão cadastrados, quantas revisões estão pendentes, ultimos serviços realizados e permite acesso rápido as outras páginas.

#### Lista de Veículos
Oferecer controle completo sobre os veículos já cadastrados.

#### Cadastro de Veículo
Permitir registrar um novo veículo ou editar os dados existentes.

#### Registrar Revisão
Registrar cada serviço realizado.

#### Relatórios
Permite acompanhar os custos com manutenção e analisar o histórico de revisões.

#### Tudo está sucetivél à mudanças durante o desenvolvimento caso acha necessidade.
